MC3 Language N/A

Check the language of the webpage is set correctly

  • To check this, right click and view Page Source. At the top of the code there should be a piece of code set as the following: <html lang="en-GB"> or <html lang="en"> The entry specifies the natural language of the content on the webpage. This helps screen readers to know what language settings to use.
